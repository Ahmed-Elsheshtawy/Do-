# To-Do List App

A simple and clean to-do list web application for managing daily tasks.

## Features

- **Add Tasks**: Enter new tasks using the text input field
- **Task Organization**: View all your tasks in an organized list
- **Sidebar Navigation**: 
  - Tasks view
  - Important tasks
  - Completed tasks
- **Clean UI**: Modern interface with custom icons and styling

## Project Structure

- `sourceCode/index.html` - Main HTML structure
- `sourceCode/styles.css` - Styling and layout
- `sourceCode/script.js` - JavaScript functionality
- `sourceCode/img/` - Icons and images

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

Simply open `sourceCode/index.html` in your web browser to start using the app.
